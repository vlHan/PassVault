<h1>Contributing </h1>

How can I contribute to the project?
```sh
1. Create a fork from PassVault repository
2. git clone https://github.com/vlHan/PassVault.git
3. cd PassVault/
4. Make your changes.
5. Commit and make a git push.
6. Open a pull request.
```
